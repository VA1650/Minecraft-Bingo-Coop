# 🎲 Minecraft Bingo Coop

Простой и надежный плагин для игры в "Бинго" в Minecraft. Разработан специально для кооперативной игры с автоматизацией сброса мира.

## 🚀 Особенности

* **Гибкий выбор целей:** Можно выбирать предметы вручную из креатива или генерировать случайный список.
* **Умная фильтрация:** Рандомайзер исключает технические блоки.
* **Защита от ошибок:** Встроены проверки на повторный запуск, пустые списки целей и ввод текста вместо чисел.
* **Глобальный скорборд:** Все игроки видят общий прогресс в реальном времени.
* **Авто-сброс мира:** При победе игра останавливается, а при следующем запуске сервера мир полностью обновляется.

## 🛠 Команды

| Команда | Описание |
| :--- | :--- |
| `/bingo pick` | Переход в режим выбора. |
| `/bingo random <кол-во>` | Генерация случайных целей. |
| `/bingo start` | Начало игры. |
| `/bingo stop` | Принудительная остановка и сброс настроек. |
| `/bingo list` | Посмотреть список целей в чате. |

## ⚙️ Технические детали

* **Версия:** Paper/Spigot 1.20.x и выше
* **API:** Paper API, Adventure API 
* **Ядро:** Java 17+

### Скрипт для авто-удаления мира (`start.bat`):
Для работы механики пересоздания мира используйте этот скрипт запуска:
```batch
@echo off
rd /s /q "world"
rd /s /q "world_nether"
rd /s /q "world_the_end"
java -Xmx2G -jar your-paper-file.jar nogui
pause
```
## 📝 Как установить

*    Скомпилируйте проект в .jar файл.
*    Поместите его в папку plugins вашего сервера.
*    Запустите сервер и наслаждайтесь игрой!
